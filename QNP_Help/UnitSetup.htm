Unit Setup

User has access only to all white fields in this form.



UnitSetup form allows user to set base parameters of an aircraft in database .

Any fuel value reached below set 'MinFuel' value will be shown in red colour in FlightPlan form.

Airspeed /fuel consumtions parameters will be updated from databse in to FlightPlan form .
User can over ride these default parameters with new parameters in FlightPlan form
