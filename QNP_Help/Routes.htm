ROUTES

Route can be created in this menu by using waypoints available in database.
Waypoint not available in database will not be accepted to create routes .
Total of max 14 points can be saved in one route.
Route name (eg XYZ) given will be saved as' ROUTE_XYZ ' in the database.
In the present version , route created can not be edited .
User will have to delete the route and recreate the new route once again.
