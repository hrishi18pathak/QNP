How To Use

Start screen gives user access to various sub screens.

Air DataBase : This form allows user to view, add and amend the air data records.

Routes : This form facilitates user to create routes from the points available in Air DataBase.

QuickNavPlanner : In this form a detailed flight planning task can be worked out.

Saved FlightPlans(pdf) : To view saved flight plans.

Aircraft Configuration : Will allow user to set basic parameters like speed and fuel consumption .
