AirDataBase




AirDataBase form allows user to add, edit and delete the waypoints in database.

To use a new waypoint not availble in database, user will need to close the FlightPlan form and insert the waypoint in data base and then utilise the same in FlightPlan form.

Data Base will be updated on every 'Refresh' button press or 'Exit' event.

If same name waypoints with diferent coordinates are tried to be saved then existing waypoint is not overwritten. User needs to delete first old waypoint and then create new waypoint.
